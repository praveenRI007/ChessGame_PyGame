# ChessGame_PyGame
Chess Game in python

- Step1: Download the Repo Files
- Step2: Extract MainChess.zip
- Step3: Inside "MainChess" Extracted Folder , paste PiecesPNG folder.
- Step4: Voila ! Run the MainChess.exe and enjoy the Game ! :)
